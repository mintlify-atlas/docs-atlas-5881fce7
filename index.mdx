  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your flashcard generator running in minutes
  </p>
  
  <Steps>
    <Step title="Get a Gemini API key">
      Visit [Google AI Studio](https://aistudio.google.com/api-keys) and create a free API key. You'll need this to power the AI flashcard generation.
      
      <Note>
        The application uses the `gemini-2.5-flash-lite` model for fast, efficient flashcard generation.
      </Note>
    </Step>
    
    <Step title="Install dependencies">
      Clone the repository and install the required Python packages:
      
      ```bash
      git clone https://github.com/errwnd/minimal-FlashCard-Python.git
      cd minimal-FlashCard-Python
      pip install -r requirements.txt
      ```
    </Step>
    
    <Step title="Configure your API key">
      Create a `.env` file in the project root and add your API key:
      
      ```bash
      API_KEY=your_gemini_api_key_here
      ```
      
      Alternatively, you can enter the API key directly in the web interface when uploading files.
    </Step>
    
    <Step title="Start the server">
      Launch the FastAPI server:
      
      ```bash
      uvicorn main:app
      ```
      
      Open your browser to `http://localhost:8000` and start generating flashcards from your PDFs.
    </Step>
  </Steps>
</div>
