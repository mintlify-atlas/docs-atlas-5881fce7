## Prerequisites

Before you begin, ensure you have:

- **Python 3.7+** installed on your system
- **Gemini API key** from [Google AI Studio](https://aistudio.google.com/api-keys)

<Note>
Your Gemini API key is required for the AI flashcard generation feature. You can provide it through the web interface or set it as an environment variable.
</Note>

## Installation

<Steps>
  <Step title="Clone or download the source code">
    Navigate to your workspace directory where the FlashCard application is located:
    
    ```bash
    cd ~/workspace/source/
    ```
  </Step>

  <Step title="Install dependencies">
    Install all required Python packages using pip:
    
    ```bash
    pip install -r requirements.txt
    ```
    
    This installs:
    - FastAPI and Uvicorn for the web server
    - PyPDF2 for PDF processing
    - LangChain and Google Gemini integration
    - Additional utilities
  </Step>

  <Step title="Configure your API key (Optional)">
    You can set up your Gemini API key as an environment variable. Create a `.env` file in the project root:
    
    ```bash
    API_KEY=your_gemini_api_key_here
    ```
    
    <Info>
    If you skip this step, you can still provide your API key through the web interface when generating flashcards.
    </Info>
  </Step>

  <Step title="Start the application">
    Launch the FastAPI server using Uvicorn:
    
    ```bash
    uvicorn main:app
    ```
    
    The server will start on `http://127.0.0.1:8000` by default.
    
    <Info>
    Add `--reload` flag for development mode: `uvicorn main:app --reload`
    </Info>
  </Step>

  <Step title="Access the web interface">
    Open your browser and navigate to:
    
    ```
    http://127.0.0.1:8000
    ```
    
    You'll see the PDF to Flashcards interface.
  </Step>
</Steps>

## Generate Your First Flashcards

Once the application is running, follow these steps to create your first set of flashcards:

<Steps>
  <Step title="Enter your API key">
    In the web interface, enter your Gemini API key in the provided field. The key is stored locally in your browser for convenience.
  </Step>

  <Step title="Upload a PDF">
    Click the file input and select a PDF document from your computer. Any PDF with extractable text will work.
  </Step>

  <Step title="Generate flashcards">
    Click the "Generate Flashcards" button. The application will:
    
    - Upload and process your PDF
    - Extract text content (up to 15,000 characters)
    - Send the content to Gemini AI
    - Parse and display the generated flashcards
    
    <Note>
    The Gemini model (`gemini-2.5-flash-lite`) will generate up to 20 flashcards from your document.
    </Note>
  </Step>

  <Step title="Review your flashcards">
    Your flashcards will appear below the upload form in the format:
    
    ```
    Q: What is the main topic? | A: The main topic is...
    ```
    
    Each flashcard is numbered and displayed in an easy-to-read card format.
  </Step>
</Steps>

## Understanding the Code

The flashcard generation happens in the `/upload` endpoint in `main.py:24-64`. Here's how it works:

```python
@app.post("/upload")
async def upload(file: UploadFile = File(...), api_key: str = Form(None)):
    # Use provided API key or fall back to environment variable
    key = api_key or API_KEY
    if not key:
        return {"error": "API key is required"}

    # Extract text from PDF
    text = ""
    with open(tmp_path, "rb") as f:
        reader = PyPDF2.PdfReader(f)
        for page in reader.pages:
            text += page.extract_text() or ""

    # Generate flashcards using Gemini
    llm = ChatGoogleGenerativeAI(model="gemini-2.5-flash-lite", api_key=key)

    prompt = f"""Create 20 simple flashcards from this text.
Format each as: Q: question | A: answer
One flashcard per line. Keep it simple.

Text:
{text[:15000]}"""

    response = llm.invoke(prompt)
```

<Warning>
The application processes only the first 15,000 characters of extracted text to stay within API limits. For longer documents, consider splitting them into smaller PDFs.
</Warning>
